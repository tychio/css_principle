CSS Guide

  • 分离皮肤和结构

    Sparate structure from skin.

    不要将皮肤样式和结构样式放在同一class中，比如一个带icon的button，应该分类2个class，icon和button，一个只有icon的图片，一个只有button的尺寸:

    .icon { 
	background-image: url(...);	}
.button {
	width: 80px;
	height: 40px; }
.button .icon {
	width: 20px;
	height: 20px;}

  • 避免依赖元素结构

    Avoid depend on structure of element.

    不要依赖于元素结构，比如子代选择器，当结构发生变化时，选择器很容易失效：

    .parent > child {
	...;
}

  • 将样式分为3个类型：基础、模块、状态

    Three categories of styles is Base, Module and State.

  • 分组特殊的属性并添加选择器

    Group special property and append selectors.
