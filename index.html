Don’t Repeat Your CSS
The 2 principles for writing good css

Keep style separate from content.

Avoid specificity by harnessing the cascade.

DRY CSS comes down to 3 things

Group reusable css properties together.

Name these groups logically.

Add your selectors to the various css groups.

Additional Thoughts

It encourages you to think in terms of style patterns (groups) rather than individual objects.

The naming of groups encourages rational organization of design.

It encourages optimization and generalization of selectors.

It doesn’t require changes to html.

There shouldn’t be any performance issues.
